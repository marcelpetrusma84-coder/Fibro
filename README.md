# Fibro

**Fibro** is an invite-only, privacy-first social platform where profiles are personal spaces, timelines are optional, and users control how they express themselves.

Instead of algorithmic feeds and engagement pressure, Fibro focuses on calm interaction, customization, and small trusted networks.

---

## What is Fibro?

Fibro is inspired by early social networks where a profile felt like a personal website rather than a performance stage.

Each user has their own customizable space:
- A profile you can design freely
- Optional timelines (chronological, non-algorithmic)
- Media, mood, and personal widgets
- A Guestbook for short personal messages

Fibro is not designed for virality or public discovery.  
It is designed for **meaningful, private connections**.

---

## Core Principles

- **Profile-first** — the profile is the center, not a global feed  
- **Optional timelines** — never forced, never algorithmic  
- **Invite-only by design** — trust-based networks  
- **Privacy-first** — users own their content  
- **Customization as expression** — few limits, many possibilities  

---

## Key Features

### Profiles
- Fully customizable, block-based layouts
- Themes for colors, fonts, backgrounds, and mood
- Optional templates for less creative users

### Blocks
Possible profile blocks include:
- Timeline (optional)
- Photo galleries
- Music and video players
- Mood or status widgets
- Friends list
- QR profile link
- Guestbook

### Guestbook
A profile-centered space where visitors can leave short personal messages.
- Chronological
- Calm and non-algorithmic
- Separate from chat and timelines

### Interaction
- Likes on posts
- Private reactions
- 1-on-1 chat
- Voice and video calls (planned)

### Safety
- External links are copyable but not clickable
- Invite-only access
- No public discovery feeds
